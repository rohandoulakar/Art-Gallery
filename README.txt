Art Gallery Project
Name: Rohan Doulakar

First install all the dependencies with the command: npm install

Next to run the server by using this command: node index.js

Artwork data is stored using mongoose in the folder “db” and in the folder “models” it contains
all the schemas.

Login and Password of “master” will provide the initial gallery

Usage: Interact with the webpage by using this url: http://localhost:3000 in the browser
